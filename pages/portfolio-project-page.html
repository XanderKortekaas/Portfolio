Building my professional portfolio

This page details the development of my portfolio website.
This project is started as a practical learing opportunity, so that i can learn Html and css.

The Process

Initially, my goal was to create a simple yet effective showcase of my skills and projects.
This motivated me to rebuild my portfolio as a practical opportunity to learn HTML and CSS.
The development process wasn't without its challenges, as is common with any project.
I encountered an issue where the placement of elements wasn't behaving as expected.
After a few days of debugging, I discovered that the site's maximum width was the constraint.
By doubling its original size, the layout issues were resolved.

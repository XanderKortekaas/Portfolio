Building a game in Godot

This page details the beginnings of my Godot project, with the reasons and objective for begining this project

The Process

I started with a simple game idea for Godot and knew planning was key. So, I brainstormed first.
To create a Game Design Document (GDD), I first decided what kind of game I wanted to make (genre and scope). Then, I chose between 2D or 3D. Next, I figured out the main gameplay mechanics.
Before opening Godot, I wrote down all my initial ideas. This pre-planning helped me have a clear vision and avoid just making random stuff. I wanted a well-thought-out design to follow.

Technologies used.

  • Godot
  • GDscript

Key learnings

  • How Godot works
  • How you code in Godot
